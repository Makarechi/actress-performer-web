# casting_profile.md

## Purpose
Casting profile is the practical page for professionals. It should be fast, clear and almost boring in the best way.

No long poetic texts. No heavy animation. No hidden information.

## Recommended URL
```txt
/en/casting/
/sk/casting/
```

## Page structure

### 1. Header
- Name
- Main profession
- Contact button
- Download CV button

### 2. Top block
Left:

- neutral headshot;
- contact details;
- base location;
- work availability.

Right:

- short intro;
- showreel;
- download CV;
- language list.

### 3. Facts
Fields to include if available:

- height;
- hair color;
- eye color;
- playing age range;
- base location;
- passports / work permissions, only if relevant and safe;
- driving licence, if relevant;
- union / representation, if relevant.

Do not invent missing facts. Leave optional fields out if not needed.

### 4. Languages
Display with use cases:

```txt
Slovak — fluent — acting / hosting / voice
Polish — fluent — acting / voice
Italian — fluent — acting / voice / singing
Ukrainian — fluent — acting / voice
Russian — fluent — acting / voice / hosting
English — fluent — acting / voice / international work
Hungarian — basic / conversational
```

### 5. Acting credits
Use compact table:

| Year | Project | Type | Role | Director/Production |
|---|---|---|---|---|
|  |  | Theatre/Film/Series/Ad |  |  |

### 6. Voice / dubbing
- short note;
- 3–5 audio samples;
- link to full voice page.

### 7. Special skills
Group skills:

**Performance:** acting, voice, hosting, dubbing.  
**Music:** vocal, piano.  
**Movement:** aerial gymnastics.  
**Languages:** listed separately.  
**Creative:** drawing.  
**Professional:** translation, event moderation.

### 8. Downloads
- CV EN;
- CV SK;
- headshot pack;
- press kit if relevant.

### 9. Contact
- email;
- phone/WhatsApp if public;
- agent/manager if exists;
- social links.

## Design rules
- No autoplay.
- No scroll-jacking.
- No giant gallery.
- No vague text.
- Print/PDF-friendly layout.
- Mobile must be excellent.
