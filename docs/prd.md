# prd.md

## Product Requirements Document

## 1. Overview
Создать персональный сайт для актрисы и мультиязычной performer, который объединяет acting, voice, languages, music, hosting и special skills в одну профессиональную платформу.

## 2. Problem
У артистки много направлений и сильных навыков. Без структуры это может выглядеть хаотично: актриса, певица, ведущая, переводчик, художница, гимнастка.

Проблема сайта — не показать всё подряд, а собрать всё в понятный профессиональный образ.

## 3. Solution
Сайт должен иметь две логики:

1. Эмоциональную: создать впечатление сильной, красивой, международной артистки.
2. Практичную: дать кастинг-директору / агенту / клиенту быстрый доступ к showreel, voice, languages, CV и contact.

## 4. Target Users
### Casting Director
Нужны:

- фото;
- параметры;
- showreel;
- опыт;
- языки;
- special skills;
- контакты.

### Voice / Dubbing Studio
Нужны:

- voice samples;
- языки;
- диапазон голоса;
- опыт дубляжа;
- возможность быстро связаться.

### Event Agency
Нужны:

- опыт ведущей;
- языки ведения;
- видео / фото;
- формат мероприятий;
- контакт.

### Director / Producer
Нужны:

- artistic impression;
- screen presence;
- theatre / film experience;
- special skills;
- доступность для проектов.

### Press / Collaboration
Нужны:

- short bio;
- photos;
- links;
- contact.

## 5. Core Value Proposition
> One artist, multiple professional formats: screen, stage, voice, music and live events — in several languages.

## 6. Functional Requirements

### Home
Must have:

- strong hero photo;
- name;
- positioning line;
- primary CTA: Watch Showreel;
- secondary CTA: Contact / Download CV;
- quick skill badges;
- featured media;
- language list;
- contact block.

### Acting Page
Must have:

- showreel;
- credits;
- gallery;
- acting CV download;
- special skills relevant to casting.

### Voice Page
Must have:

- audio samples;
- language filters;
- sample categories: commercial, narration, character, dramatic;
- dubbing experience;
- contact CTA.

### Music Page
Must have:

- vocal sample;
- piano sample;
- live performance note;
- musical styles if known;
- contact CTA.

### Hosting Page
Must have:

- event hosting experience;
- languages for hosting;
- event formats;
- photos / videos;
- contact CTA.

### Casting Profile
Must have:

- headshot;
- showreel;
- contact;
- physical parameters if provided;
- languages;
- credits;
- skills;
- downloadable CV;
- no unnecessary animations.

### Press Kit
Must have:

- short bio;
- medium bio;
- downloadable photos;
- CV;
- official contact;
- social links.

### Contact
Must have:

- email;
- optional phone / WhatsApp;
- location base;
- availability note;
- representation / agent if applicable;
- links to social platforms.

## 7. Non-functional Requirements
- Mobile-first.
- Fast loading.
- Accessible media controls.
- SEO-ready.
- Multilingual-ready.
- Secure contact handling.
- Easy content update.
- No autoplay audio.
- Private materials must not be indexed.

## 8. Content Requirements
Each media item should include:

- title;
- type;
- language;
- year if relevant;
- credit / project;
- thumbnail;
- external URL or local asset;
- visibility: public / private;
- priority.

## 9. Out of Scope for MVP
- blog;
- online booking calendar;
- e-commerce;
- paid content;
- full CRM;
- complex user accounts;
- social network features.

## 10. Success Criteria
- Visitor understands positioning within 10 seconds.
- Showreel is accessible above the fold or one click away.
- Contact is always visible.
- CV download works.
- Pages load fast on mobile.
- Site ranks for the artist's name.
- Site can be sent directly in casting emails.
