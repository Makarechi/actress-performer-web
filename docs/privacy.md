# privacy.md

## Privacy and rights

## Personal data
Не публиковать:

- домашний адрес;
- личные документы;
- персональные идентификаторы;
- личный номер телефона, если нет желания получать прямые звонки;
- приватные материалы без защиты.

## Contact data
Рекомендовано:

- professional email on domain;
- optional WhatsApp business number;
- contact form with spam protection;
- agent / manager contact if exists.

## Media rights
Перед публикацией проверить:

- можно ли использовать кадры из фильма / сериала;
- можно ли использовать рекламные ролики;
- можно ли публиковать backstage;
- есть ли права фотографа;
- нужно ли указывать credits.

## Private casting materials
Private материалы:

- не добавлять в sitemap;
- ставить `noindex`;
- не публиковать в соцсетях;
- не использовать очевидные URL вроде `/private/video1`;
- по возможности использовать token links.

## Downloads
CV и press kit публичны, но должны содержать только профессиональную информацию.

Не добавлять в CV:

- домашний адрес;
- дату рождения, если это не нужно рынку;
- лишние личные данные;
- документы;
- private phone, если некомфортно.

## Contact form
Если есть форма, добавить:

- privacy note;
- consent checkbox, если требует юрисдикция;
- spam protection;
- ограничение attachments, если форма принимает файлы.

## Cookies
Если используется analytics с cookies, добавить cookie/privacy notice по необходимости.

Если используется privacy-friendly analytics без cookies, текст может быть проще.
