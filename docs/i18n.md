# i18n.md

## Цель мультиязычности
Мультиязычность — один из главных коммерческих активов проекта. Она должна быть не только текстовой, но и доказательной: через voice samples, видео и language clips.

## Стартовые языки
MVP:

- English;
- Slovak.

Почему:

- English — международные кастинги, агенты, voice, реклама.
- Slovak — локальный рынок, Центральная Европа, театральные/ивент/медиа контакты.

## Следующие языки

Этап 3:

## Принцип локализации
Не переводить механически 100% текста. Лучше:

1. Полноценный английский сайт.
2. Полноценный словацкий сайт.
3. Короткие локальные landing sections для остальных языков.
4. Voice/language samples как доказательство владения языком.

## Locale structure


## Fallback logic
Если для страницы нет полного перевода:

- показывать английскую версию;
- не показывать пустую локаль;
- не использовать машинный перевод без вычитки;
- в интерфейсе не обещать язык, если страница на нём неполная.

## Tone by language
### English
Professional, international, clean.

### Slovak
Local, professional, not overly poetic.

### Russian / Ukrainian
Тёплый, но не слишком литературный. Больше конкретики.

### Polish / Italian
Лучше вычитать носителем или человеком с сильным уровнем. Ошибка в языковой версии будет бить по главному преимуществу — языкам.

## Language level labels
Использовать понятные уровни:

- Native;
- Fluent;
- Professional working proficiency;
- Conversational;
- Basic.

Не писать “fluent”, если в профессиональном контексте есть сомнения.

## hreflang
Для каждой публичной локализованной страницы добавить hreflang.

Пример:

```html
<link rel="alternate" hreflang="en" href="https://domain.com/en/acting/" />
<link rel="alternate" hreflang="sk" href="https://domain.com/sk/acting/" />
<link rel="alternate" hreflang="x-default" href="https://domain.com/en/acting/" />
```

## Content keys
Избегать строк, разбросанных по компонентам. Лучше использовать словари:

```ts
const messages = {
  en: {
    watchShowreel: 'Watch Showreel',
    contact: 'Contact',
  },
  sk: {
    watchShowreel: 'Pozrieť showreel',
    contact: 'Kontakt',
  },
};
```

## QA для языков
Перед запуском проверить:

- нет смешения языков в одном блоке;
- кнопки переведены;
- meta title и description переведены;
- alt-тексты переведены;
- PDF CV соответствует языку;
- voice samples правильно подписаны;
- переключатель языков не ведёт на пустые страницы.
