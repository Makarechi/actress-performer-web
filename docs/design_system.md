# design_system.md

## Visual direction
Сайт должен быть премиальным, минималистичным и артистичным. Не «глянцевый шаблон», не «театральный плакат», не «портфолио дизайнера».

Ключевое ощущение:

> Cinematic elegance with practical casting clarity.

## Design principles
1. Лицо и голос важнее декоративности.
2. Много воздуха.
3. Чёткая типографика.
4. Контрастные CTA.
5. Медиа показываются крупно, но не ломают скорость.
6. Casting page — максимально практичная.
7. Главная — эмоциональная, но понятная.

## Layout
### Home hero
- large portrait / cinematic image;
- name;
- one-line positioning;
- 3–4 CTA;
- language badges;
- maybe showreel preview.

### Content sections
- section title;
- short intro;
- selected media;
- one CTA;
- no huge walls of text.

### Casting profile
- left: headshot + contact;
- right: facts, showreel, CV, skills;
- printable-friendly layout;
- minimal animation.

## Typography
Рекомендация:

- elegant serif for headings or name;
- clean sans-serif for body;
- avoid overly decorative fonts;
- ensure diacritics support: Slovak, Polish, Ukrainian, Russian, Italian.

Важно: шрифт должен поддерживать кириллицу и латиницу с диакритикой.

## Color direction
Не задаю конкретные цвета без фото, но направление:

- warm neutral base;
- dark cinematic text/background option;
- one accent color;
- no rainbow palette;
- no childish pink unless это сознательный artistic choice.

## Components
### Buttons
Types:

- primary;
- secondary;
- text link;
- download;
- external.

Primary CTA examples:

- Watch Showreel;
- Contact;
- Download CV.

### Cards
Media card should show:

- thumbnail;
- title;
- category;
- language;
- duration if media;
- play/download action.

### Skill tags
Use tags, but not too many in one row.

Good:

```txt
Acting · Voice · Vocal · Piano · Hosting · Aerial · Drawing
```

Bad:

```txt
30 badges fighting for attention
```

### Audio player
Needs:

- play/pause;
- title;
- language;
- category;
- duration.

Waveform optional.

## Animation
Use carefully:

- subtle fade-in;
- image reveal;
- hover states;
- smooth scroll.

Avoid:

- heavy parallax;
- complex 3D;
- scroll hijacking;
- animations that delay content;
- motion that makes mobile uncomfortable.

## Accessibility
- visible focus states;
- text contrast;
- font size readable;
- captions where needed;
- controls for audio/video;
- no essential information only in image.

## Photo treatment
- maintain skin texture;
- avoid over-filtering;
- consistent crop ratios;
- separate headshots and artistic photos;
- no messy collage.

## Design anti-patterns
- autoplay music;
- homepage only as video background;
- navigation hidden behind experimental UI;
- long poetic intro before profession;
- showreel below 5 screens;
- 20 fonts;
- huge galleries without curation.
