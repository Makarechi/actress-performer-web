# media_guidelines.md

## Главная идея
Материалы должны доказывать профессиональность быстрее, чем текст.

Лучше мало сильных материалов, чем много средних.

## Фото
### Обязательный минимум
- 1 главный headshot;
- 1 waist-up portrait;
- 1 full-body photo;
- 2–4 acting / production stills;
- 1 elegant editorial portrait;
- 1 фото для press kit.

### Рекомендации
- не смешивать бытовые фото с профессиональными;
- не использовать слишком агрессивную ретушь;
- показать лицо ясно;
- иметь разные эмоциональные состояния;
- иметь фото с нейтральным фоном для casting profile;
- иметь cinematic фото для главной.

### Alt text examples
English:
> Portrait of [Name], multilingual actress and performer.

Russian:
> Портрет [Имя], мультиязычной актрисы и performer.

## Видео
### Acting showreel
Идеальная длина: 60–90 секунд.

Структура:

1. сильный первый кадр;
2. 3–5 разных эмоциональных состояний;
3. диалоги / крупные планы;
4. кадр из рекламы или сериала, если сильный;
5. финальный контакт / имя / сайт.

Не делать:

- длинное интро;
- логотипы 10 секунд;
- медленные титры;
- сцены, где её почти не видно;
- плохой звук.

### Music videos
- вокал: 30–90 секунд;
- фортепиано: 30–90 секунд;
- live performance: коротко и с хорошим звуком;
- лучше 2 коротких видео, чем одно на 5 минут.

### Hosting video
Нужно показать:

- уверенность;
- речь;
- контакт с аудиторией;
- языки;
- энергию;
- формат события.

## Voice samples
### Минимальный набор
- commercial sample;
- narration sample;
- character sample;
- dramatic sample;
- language samples.

### Длительность
- 20–45 секунд для language sample;
- 30–60 секунд для commercial/narration;
- до 90 секунд для dramatic.

### Для каждого sample указать
- язык;
- жанр;
- длительность;
- настроение;
- год записи, если уместно.

Пример:

```json
{
  "title": "Commercial Voice Sample",
  "language": "English",
  "category": "commercial",
  "duration": 34
}
```

## Language samples
Сильная идея: записать одну и ту же короткую фразу или похожий текст на разных языках.

Языки:

- Slovak;
- Polish;
- Italian;
- Ukrainian;
- Russian;
- English;
- Hungarian, если уровень позволяет.

## Audio quality
Минимально приемлемо:

- чистая запись без эха;
- без фонового шума;
- нормализованный уровень громкости;
- без перегруза;
- одинаковая громкость между samples.

## File naming
```txt
name_category_language_year.ext
```

Примеры:

```txt
artist_headshot_01_2026.webp
artist_showreel_2026.mp4
artist_voice_commercial_en_2026.mp3
artist_voice_character_it_2026.mp3
artist_cv_en_2026.pdf
```

## Что не публиковать
- материалы с плохим звуком;
- длинные неотобранные записи;
- видео с чужими правами без разрешения;
- private casting tapes публично;
- фото, которые не соответствуют профессиональному уровню сайта.

## Press kit assets
В press kit включить:

- 2–3 headshots;
- short bio;
- medium bio;
- CV;
- social links;
- contact;
- selected credits;
- maybe 1 official poster / production still if rights allow.
